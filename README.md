# Лабораторная работа №1: Геометрические вычисления (Трапеция)
**Вариант №30**

## Описание проекта
Консольное приложение на языке C#, предназначенное для автоматизации расчетов параметров геометрической фигуры — трапеции. Программа позволяет вычислять площадь, среднюю линию и периметр равнобедренной трапеции на основе вводимых пользователем данных.

### Основной функционал:
1. **Расчет площади**: Вычисление по формуле $S = \frac{a + b}{2} \cdot h$.
2. **Расчет средней линии**: Определение расстояния между серединами боковых сторон по формуле $m = \frac{a + b}{2}$.
3. **Расчет периметра**: Нахождение периметра равнобедренной трапеции с использованием теоремы Пифагора для поиска боковых сторон.
4. **Меню управления**: Интерактивный интерфейс для выбора необходимых операций.

---

## Особенности реализации
*   **Валидация данных**: Реализован метод `ReadPositiveDouble`, который исключает ввод некорректных символов (букв) и отрицательных чисел.
*   **Математический аппарат**: Использование библиотеки `System.Math` для вычисления квадратных корней и возведения в степень.
*   **Циклический интерфейс**: Программа работает в бесконечном цикле `while` до тех пор, пока пользователь явно не выберет пункт выхода.

---

## Алгоритм расчета периметра
Для нахождения периметра равнобедренной трапеции программа сначала вычисляет длину боковой стороны $c$ по формуле:
$$c = \sqrt{h^2 + \left(\frac{a - b}{2}\right)^2}$$
После чего вычисляется итоговый периметр: $P = a + b + 2c$.
